# Joy-Con Flight System for PC

Nintendo SwitchのJoy-Conを2本使い、ジャイロ操作で直感的にフライトシミュレーター（MSFS2020等）を操作するためのPythonシステムです。

## 特徴
- **ジャイロ操縦**: 左コンを傾けてロール・ピッチを操作。
- **カメラ操作**: 右コンを傾けて自由な視点移動。
- **デジタル・スムージング**: L1/L2ボタンによる滑らかなラダー（ヨー）操作。
- **vJoy連携**: 仮想ジョイスティックとして認識されるため、多くのゲームに対応。

## セットアップ

### 1. 必要なソフト
- [vJoy](https://github.com/shmdn/vJoy/releases) のインストール
- Python 3.x

### 2. ライブラリのインストール
```bash
pip install hidapi pyvjoy
```
## ⚙️ パーソナライズ（数値の書き換え）

Joy-Conの個体差により、ジャイロやスティックの基準値は異なります。より正確な操作のために、以下の手順で数値を自分のものに書き換えてください。

### 1. ジャイロのオフセット設定
1. `joycon_Gyro.py` を実行し、画面の指示に従ってJoy-Conを静止させます。
2. 表示された `OFFSETS = { ... }` の内容をコピーします。
3. `joycon_vjoy_flyght.py` 内の `OFFSETS` 変数（14行目付近）に貼り付けます。

### 2. スティックのキャリブレーション
1. `joycon_analog.py` を実行し、スティックを上下左右に動かして数値を記録します。
2. 得られた中心値（C）や最大値（U, D, L, R）を `joycon_vjoy_flyght.py` 内の `CALIB` 変数（19行目付近）に反映させてください。
